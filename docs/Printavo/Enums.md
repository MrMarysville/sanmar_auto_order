# Enums

Enums represent a set of possible values for a field.

Examples of enums:
- `TaskableType`
- `LineItemSize`
- `LineItemStatus`
- `MerchOrderStatus`
- `MerchStoreStatus`
- `MessageDeliveryStatus`
- `OrderPaymentStatus`
- `OrderSortField`
- `PaymentDisputeStatusField`
- `PoGoodsStatus`
- `StatusType`
- `TaskSortField`
- `TransactionCategory`
- `TransactionSource`

For more information, see [the GraphQL spec](https://facebook.github.io/graphql/#sec-Enums).
