# Unions

Unions are types of objects that can represent one of many different object types.

Examples of unions:
- `TransactedForUnion`
- `OrderUnion`
- `MessageParticipantUnion`
- `TaskableUnion`
- `TransactionUnion`
- `MessageUnion`

For more information, see [the GraphQL spec](https://facebook.github.io/graphql/#sec-Unions).
